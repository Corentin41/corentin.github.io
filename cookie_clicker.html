Cookie Clicker

Contexte

J'ai réalisé ce projet au cours de mon année de Licence Informatique dans le cadre d'un cours de Programmation Mobile. Il n'y avait pas de sujet imposé. Toutefois, il fallait intégrer certaines choses dans notre application comme le stockage des informations dans une base de données locale (intégrée dans le téléphone, sans passer par internet), les capteurs du téléphone Il fallait aussi prévoir un mode connecté et un mode non connecté.

Objectifs

Pour ce projet, nous avons fait le choix de reprendre le jeu web "Cookie Clicker". Le principe est assez simple. Il suffit de taper sur un cookie pour obtenir encore plus de cookies. L'objectif final étant d'avoir plus de cookies que les autres joueurs. Développement d'une application mobile Android avec le langage de programmation Java.

Bilan

Ce projet a consitué mon premier projet mobile. Suite à ce projet, je me suis découvert une attirance pour le domaine du développement d'applications mobiles.

Plusieurs contraintes ont été imposées par l'enseignant (internationalisation, utilisation de fragments, utilisation d'un service, mode d'affichage en portrait et en paysage...).
